# Sistema de Biblioteca

Projeto desenvolvido em Java com o objetivo de praticar os principais conceitos de Programação Orientada a Objetos (POO).

## Objetivo

Simular o funcionamento básico de uma biblioteca, permitindo o cadastro de livros e usuários, além da realização de empréstimos e devoluções.

## Funcionalidades

* Cadastro de livros
* Cadastro de usuários
* Empréstimo de livros
* Devolução de livros
* Controle de quantidade disponível
* Registro da data do empréstimo
* Registro da data da devolução
* Tratamento de exceções
* Uso de interfaces e classes abstratas

## Conceitos praticados

* Orientação a Objetos
* Encapsulamento
* Herança
* Polimorfismo
* Classes abstratas
* Interfaces
* Collections (`List`)
* Exceções personalizadas
* Organização em pacotes
* API `java.time`

## Tecnologias utilizadas

* Java
* Eclipse IDE

## Como executar

1. Clone o repositório.
2. Abra o projeto na sua IDE Java.
3. Execute a classe `Main`.
4. O sistema exibirá os livros cadastrados, os empréstimos realizados e as devoluções.
